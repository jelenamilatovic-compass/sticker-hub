# Sličica Hub ⚽

Online hub za razmjenu sličica iz albuma — djeca unose šta imaju duplo i šta im fali, hub automatski nađe ko se s kim može mijenjati.

## Funkcije
- 📝 Registracija sa imenom + listama (imam duplo / fali mi)
- 👥 Pregled svih igrača
- 🤝 Auto-matching trampi (1-za-1)
- 🔥 Trending — najtraženije sličice
- 💎 Trending — najčešće duplo

## Tech
- Cloudflare Pages (static + Functions)
- KV za podatke
- Bez build-a, čist HTML/JS/CSS
